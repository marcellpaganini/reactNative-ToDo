## Getting Started  
### Prerequisites
* [Node.js](https://nodejs.org/en/ "Download Node.js 16.15.0 LTS")  

### Project Setup (VS Code)
* React Native 
  * Create React Native App  
  ```bash
  npx create-expo-app ProjectName -t expo-template-blank-typescript
  ```    
  * Start Development Server   
  ```bash
  npx expo start
  ```
